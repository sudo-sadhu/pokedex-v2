# ⚡ Pokédex V2 Accessibility ⚡

Welcome to **Pokédex V2 Accessibility**, a modern, blazing-fast Pokédex application focused on high accessibility (WCAG compliance, screen-reader friendliness, and full keyboard navigation) built on Next.js 15+ and Tailwind CSS.

This is an accessibility-oriented version of the Pokédex project designed to ensure that everyone, including users with visual, auditory, motor, or cognitive disabilities, can search, filter, and explore the Pokémon database.

## 👁️ Accessibility Highlights
* **Screen Reader Support**: Fully integrated with semantic HTML elements and ARIA roles.
* **Keyboard Navigation**: Accessible navigation across the application without relying on a mouse.
* **Color Contrast & Dark Mode**: Handcrafted themes to support users with low vision or colorblindness.

## 🛠️ Tech Stack
* **Framework**: Next.js 15+ (App Router)
* **Language**: TypeScript
* **Styling**: Tailwind CSS

## 🚀 Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/sudo-sadhu/pokedex-v2-accessibility.git
   cd pokedex-v2-accessibility
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```
