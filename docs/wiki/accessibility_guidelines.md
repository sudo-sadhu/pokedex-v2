# Accessibility and Contribution Guidelines

## Open Source Contribution
We welcome contributions to Pokedex-v2! To ensure accessibility for everyone:
1. Always add `aria-labels` to interactive elements.
2. Maintain high contrast ratios, especially in the upcoming Dark Mode.
3. Keep semantic HTML structure.

## Process
1. Fork the repository.
2. Create your feature branch.
3. Ensure you follow our accessibility standards.
4. Submit a Pull Request.
